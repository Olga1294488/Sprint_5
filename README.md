## Структура проекта
- `tests/` - директория с тестами
  - `test_registration.py` - тесты регистрации
  - `test_login.py` - тесты входа в систему
  - `test_profile.py` - тесты личного кабинета
  - `test_constructor.py` - тесты конструктора бургеров
- `locators.py` - файл с локаторами элементов
- `data.py` - тестовые данные и генераторы
- `conftest.py` - фикстуры pytest
- `requirements.txt` - зависимости проекта
