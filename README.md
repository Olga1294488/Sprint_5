# Проект автоматизации тестирования Stellar Burgers

## Описание проекта
Автоматические тесты для веб-приложения **Stellar Burgers** — космического фастфуда, где можно собрать и заказать бургер из необычных ингредиентов.

**Сайт:** [Stellar Burgers](https://stellarburgers.education-services.ru/)

## Технологии
- **Python 3.11+**
- **Selenium WebDriver** — автоматизация браузера
- **pytest** — фреймворк для тестирования
- **webdriver-manager** — автоматическое управление драйверами
